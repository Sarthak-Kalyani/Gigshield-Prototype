# Gigshield-Prototype

GigShield is a prototype web application designed to demonstrate an AI-powered micro-insurance solution for gig workers such as delivery riders and platform workers.

The prototype shows how workers can enroll in insurance coverage, monitor risk zones, simulate trigger events, and receive automated payouts based on environmental disruptions such as flood alerts, extreme rainfall, and severe air quality conditions.

## Features

* Interactive mobile-style UI prototype
* Dashboard showing active policy details
* Onboarding flow for policy activation
* Trigger simulation for disruption events
* Automatic payout approval demo
* Policy details and payout history

## Core Use Case

GigShield is built to solve a major problem faced by gig workers:

When weather conditions, pollution, floods, or disruptions stop workers from earning, traditional insurance does not respond quickly enough.

GigShield demonstrates a system where:

* Risk is monitored using external APIs
* Trigger events are detected automatically
* Fraud checks are performed
* Payout is processed instantly to worker UPI

## Trigger Sources Simulated

* IMD Weather Alerts
* NDMA SACHET Flood Alerts
* AQI Monitoring APIs
* GPS Inactivity Verification
